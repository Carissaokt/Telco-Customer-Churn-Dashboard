🎯 Selamat Datang!

Dashboard ini dirancang untuk membantu menganalisis dan memprediksi kemungkinan pelanggan Telco yang akan melakukan churn (berhenti berlangganan). Menggunakan Machine Learning dan Logistic Regression, sistem kami dapat memberikan insight berharga untuk retention strategy.

📋 Fitur Dashboard:

  • Dashboard Analisis: Visualisasi data pelanggan Telco dengan berbagai metrik penting
  • Prediksi Churn: Prediksi apakah pelanggan akan churn berdasarkan profil mereka
  • Feature Importance: Lihat atribut apa yang paling mempengaruhi churn
  • Real-time Analysis: Analisis data secara real-time dengan model yang terlatih

💡 Bagaimana Menggunakan Dashboard?

1. 📊 Jelajahi Dashboard

Kunjungi halaman Dashboard untuk melihat visualisasi data pelanggan Telco. Anda akan menemukan chart tentang churn distribution, kontrak type, tenure, dan berbagai metrik penting lainnya.

2. 🔮 Buat Prediksi

Di halaman Prediksi, isi form dengan data pelanggan. Sistem akan menggunakan 10 atribut paling penting untuk memprediksi apakah pelanggan akan churn atau tidak.

3. 📈 Analisis Hasil

Lihat hasil prediksi beserta probabilitas churn. Semakin tinggi probabilitas, semakin besar risiko pelanggan untuk berhenti berlangganan.
